*********************************************
Osushi
*********************************************

Osushi is a multi player action game at the Merry-go-round sushi store in Japanese.

Players must get some variety of sushies moving on belt conveyor.

This game played on web browser, because It's written by JavaScript.

Game source code is written by CoffeeScript and server side's code is written by Node.js.

This game was created by Global Game Jam in Sapporo at 2012.
